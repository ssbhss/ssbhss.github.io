MyBatisPlus CURD

MyBatisPlus 是一套用于简化MyBatis操作的工具

1. 引入坐标

<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-boot-starter</artifactId>
    <version>3.5.3.1</version>
</dependency>

2. 定义实体类

使用MyBatisPlus的注解对参数进行标注，

@TableName 对应数据库表名

@TableId 对应主键

@TableField 对应属性，value为名字，select为是否查询

@TableLogic 对应逻辑删除

@Version对应乐观锁

@Data
@TableName("tbl_book")
public class Book {

    @TableId(type = IdType.AUTO)
    Integer id;
    @TableField(value = "type", select = false)
    String type;
    @TableField(value = "type")
    String name;
    @TableField(value = "type")
    String description;
    @TableLogic(value = "0", delval = "1")
    int deleted;
    @Version
    int version;
}

3. 定义Dao层接口

通过继承BaseMapper可以获得MyBatisPlus定义好的诸多方法，在SpringBoot中注意使用@Mapper来标记这是一个Dao层操作

@Mapper
public interface BookDao extends BaseMapper<Book> {

}

4. 定义service层

service层可以使用若干MyBatisPlus定义好的方法完成基础的增删改查操作。其中值得注意的是，如何设置拦截器完成分页查询、加乐观锁等。如何使用Wrapper类进行条件查询、查询映射等

public interface BookService {
    boolean doSave(Book book);

    List<Book> selectAll();

    List<Book> selectByPages(long current, long size);
}
@Service
public class BookServiceImpl implements BookService {

    @Autowired
    BookDao bookDao;
    @Override
    public boolean doSave(Book book) {
        return bookDao.insert(book) > 0;
    }

    @Override
    public List<Book> selectAll() {

//        qw.lambda().gt(Book::getId,5);

//        方式三

//        lqw.gt(Book::getId,5);

//        LambdaQueryWrapper<Book> lqw = new LambdaQueryWrapper<Book>();
//        lqw.gt(Book::getId,5).lt(Book::getId,10).or().gt(Book::getId,11);

//        可以使用条件判断
//        LambdaQueryWrapper<Book> lqw = new LambdaQueryWrapper<Book>();
//        Integer idMax; //模拟id最大值，通常单独封装
//        idMax = 10;
//        lqw.gt(null != idMax, Book::getId,5);
//        return bookDao.selectList(lqw);

//        -----------
//        查询投影
//        方法一

//        return bookDao.selectList(lqw);

//        方法二
        QueryWrapper<Book> qw = new  QueryWrapper<Book>();
        qw.select("count(*) as count", "name")
          .groupBy("name");
        List<Map<String, Object>> maps = bookDao.selectMaps(qw);
        System.out.println(maps);
        return bookDao.selectList(qw);

    @Override
    public List<Book> selectByPages(long current, long size) {
        IPage page = new Page(current, size);
        bookDao.selectPage(page,null);
        return page.getRecords();
    }
}

5. 定义拦截器

@Configuration
public class MpConfig {

    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor(){
        MybatisPlusInterceptor mybatisPlusInterceptor = new MybatisPlusInterceptor();
        mybatisPlusInterceptor.addInnerInterceptor(new PaginationInnerInterceptor());
        mybatisPlusInterceptor.addInnerInterceptor(new OptimisticLockerInnerInterceptor());
        return mybatisPlusInterceptor;
    }
}

6. 总结

MyBatisPlus是一套简化开发的工具，合理使用使得开发变得更加简单。
